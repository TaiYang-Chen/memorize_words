> 本次已按 `Activity / Fragment / Service / Worker / Receiver / Navigation / Domain Facade / Startup Task / Sync Outbox` 逐项核对。  
> 结论以“代码当前真实行为”为准，不以 UI 文案、设计稿或历史描述为准。

## 2. 项目功能总览

这是一个围绕“词书 + 学习计划 + 正式学习 + 自由练习 + 统计 + 悬浮复习 + 账号体系 + 反馈 + 同步”构成的 Android 多模块英语学习应用。

| 一级能力 | 核心内容 | 当前状态 |
|---|---|---|
| 启动与全局任务 | 冷启动分流、会话状态预热、被踢下线监听、前台同步触发、前台词书更新检测、首屏后自动拉起悬浮复习 | 已实现 |
| 首次引导 Onboarding | 选词书、搜词书、设置学习计划、完成后自动下载并设为当前词书 | 已实现 |
| 登录注册 | 密码登录、注册、短信验证码登录、微信登录、QQ 登录 | 已实现 |
| 首页工作台 | 今日学习概览、同步 Banner、开始学习、追加新词、进入词书 | 已实现 |
| 词书管理 | 当前词书、我的词书、词书商城、词表、收藏、下载/更新/取消下载 | 已实现 |
| 学习计划 | 每日新词/复习数量、测试模式、单词顺序、重置进度 | 已实现 |
| 正式学习主流程 | 单词测验、单词详情、完整词条、分享、真题练习、完成页、打卡页 | 已实现 |
| 自由练习 | 听力、跟读、拼写、音频循环、自选词、随机词、练习记录 | 已实现 |
| 统计与打卡 | 学习总量、月历、周柱状图、日详情、自动打卡、补签入口 | 已实现 |
| 个人中心 | 学习数据、收藏、反馈、关于、账号资料、退出登录 | 已实现 |
| 账号资料 | 昵称、性别、头像、密码、社交绑定、删号、手机号绑定入口 | 部分实现 |
| 反馈与关于 | 文本反馈、图片反馈、版本展示、关于页、检查更新占位 | 已实现，部分入口占位 |
| 悬浮复习 | 悬浮球、悬浮卡、字段配置、排序、透明度、详情跳转 | 已实现 |
| 词书更新 | 前台弹窗、系统通知、静默更新、稍后提醒、忽略版本、增量/全量更新 Worker | 已实现 |
| 数据同步 | 登录后引导同步、离线 Outbox、前台自动 Drain、同步状态 Banner | 已实现 |
| 下载与语音 | 通用文件下载、词书下载、TTS、跟读评测、语音缓存 | 已实现 |

## 4. 启动、路由与全局任务

### 4.1 冷启动分流

`SplashActivity` 启动后会通过 `StartupOrchestrator` 做三分流：

| 条件 | 目标 |
|---|---|
| 未登录 / 会话无效 | `AuthActivity` |
| 已登录但 Onboarding 未完成 | `OnboardingActivity` |
| 已登录且 Onboarding 完成 | `HomeActivity` |

补充说明：

- 无网络时仍会按本地认证态和本地 Onboarding 快照做路由。
- Splash 会保证最短展示时长 500ms。

### 4.2 Application 启动任务

`MemorizeWordsApplication` 在主进程启动时注册以下任务，悬浮窗独立进程会跳过这些初始化：

| 任务 | 代码 | 实际行为 |
|---|---|---|
| 会话踢下线监听 | `SessionKickoutStartupTask` | 收到踢下线事件后清任务栈并回到启动入口 |
| 首次恢复后任务 | `PostLaunchStartupTask` | 首个非 Splash 页面恢复后预热会话状态，并在延时后尝试自动拉起悬浮复习 |
| 前台同步触发 | `ForegroundSyncStartupTask` | App 首次进入前台时触发 Sync Outbox Drain |
| 前台词书更新检测 | `ForegroundWordBookStartupTask` | App 进入前台时检查当前词书更新，必要时弹前台提示或发系统通知 |

## 6. 业务功能细化

### 6.1 登录注册与首次引导

#### 6.1.1 登录注册

- 支持手机号 + 密码登录。
- 支持手机号 + 密码注册。
- 支持发送短信验证码并做倒计时，验证码登录成功后结束认证栈。
- 支持微信 OAuth 登录，依赖 `WXEntryActivity` 接收回调。
- 支持 QQ 授权登录。
- 登录失败会做本地必填校验，并根据错误类型展示更具体提示。

#### 6.1.2 首次引导 Onboarding

选词书页能力：

- 分类 Tab 切换词书。
- 关键字搜索，带 300ms 防抖。
- 搜索模式与普通浏览模式切换。
- 空关键字提示、无结果提示、加载失败重试、下拉刷新。
- 选中词书后进入下一步；未选中时“下一步”不可用。

学习计划页能力：

- 设置每日新词数，范围 `1~80`。
- 设置每日复习数，范围 `1~160`。
- 加减按钮支持长按连续步进。
- 支持“智能平衡建议”，直接回填推荐值 `15 / 30`。
- 支持三种测试模式：释义选择、拼写、听力。
- 支持五种单词顺序：随机、字母升序、字母降序、长度升序、长度降序。
- 支持返回重新选书。

完成 Onboarding 时的真实动作：

- 保存学习计划。
- 下载所选词书，且显式使用非前台下载模式，避免退出引导页时的前台服务异常。
- 将所选词书设为当前词书。
- 将 Onboarding 快照标记为已完成。
- 将 Onboarding 状态写入 Sync Outbox，等待同步到服务端。

### 6.2 首页与统计

#### 6.2.1 首页工作台

- 展示当前用户信息。
- 展示当前词书信息。
- 展示累计学习天数、今日学习时长、今日新词完成数、今日复习完成数。
- 根据学习计划计算今日进度百分比。
- 展示同步 Banner，状态来自 Sync Outbox：隐藏、待同步、离线、阻塞。
- “开始学习”会优先进入新词学习；若今日新词已达标，则弹出追加新词对话框。
- 追加新词支持 `5 / 10 / 20` 个选项。
- 支持直接发起复习学习。
- 首页支持“双击返回退出应用”。

#### 6.2.2 练习首页

- 展示今日练习时长、总练习时长、连续练习天数、近 7 日增长比例。
- 展示最近练习记录，并可查看单次明细。
- 提供 4 个练习入口：听力、跟读、拼写、音频循环。
- 进入练习前会校验是否有当前词书、是否有可练习词。
- 支持悬浮复习总开关；切换后会直接发送服务启动或停止动作。

#### 6.2.3 统计页

- 展示总学习天数、总学习时长、总学习单词数。
- 提供周单词统计柱状图。
- 提供周时长统计柱状图。
- 周单词统计支持 `全部 / 新词 / 复习` 三个筛选维度。
- 提供月历分页视图，可切前月、后月、回到今天。
- 点击某一天可打开日详情 BottomSheet。
- 业务日期每分钟刷新一次，当前日期变更时会自动同步选中态与展示月份。

### 6.3 词书、计划与词书更新

#### 6.3.1 学习计划与词书管理

- 学习计划页展示当前词书卡片与当前学习计划。
- 支持打开词表页。
- 支持进入“我的词书”。
- 支持修改每日新词和复习数量。
- 支持切换正式学习测试模式。
- 支持切换正式学习单词顺序。
- 支持重置当前词书学习进度，且有二次确认。

#### 6.3.2 我的词书

- 展示“我的词书”列表与进度。
- 支持 `All / Studying / Completed` 过滤。
- 支持切换当前词书。
- 支持进入词书商城。
- 支持感知当前词书更新 UI 状态。

#### 6.3.3 词书商城

- 支持分类查询。
- 支持关键字搜索，带 300ms 防抖。
- 支持分页加载。
- 每本词书可显示下载状态：未下载、下载中、已下载、可更新等。
- 支持下载词书。
- 若当前状态是 `UpdateAvailable`，下载动作会走强制刷新。
- 支持取消下载。

#### 6.3.4 词表与收藏

- 词表页支持分页。
- 词表支持按状态过滤：全部、已掌握、已学、未学。
- 收藏页支持分页。
- 收藏页按收藏日期分组展示。

#### 6.3.5 词书更新体系

检测与触发：

- App 回到前台时检测当前词书更新。
- 进入词书页时检测并刷新更新候选。
- 若前台场景不适合直接提示，会改发本地通知。

提示与策略：

- 前台弹窗提醒。
- 词书页内更新横幅。
- 系统通知提醒。
- 支持稍后提醒。
- 支持忽略当前版本。
- 支持前台提醒开关。
- 支持静默更新开关。
- 静默更新支持网络策略，至少覆盖 `Wi-Fi only / Any network`。

更新执行：

- `CurrentWordBookUpdateWorker` 支持 `FULL` 全量刷新。
- 也支持 `DELTA` 增量刷新：删除移除词、拉取增量词页、同步词状态。
- 更新完成后会刷新本地词书版本、上报完成、发完成通知。
- 更新失败会记录失败状态并发失败通知。

### 6.4 正式学习主流程

- 通过 `LearningActivity` 承载正式学习会话。
- 正式学习会区分新词学习和复习学习两类会话。
- `WordLearningTestFragment` 负责按学习计划生成测验题。
- `WordLearningDetailFragment` 展示释义、例句、词根、近义反义、词形变化等内容。
- 学习详情页支持收藏、掌握标记、分享、进入完整词条。
- 例句内支持可点击单词，弹出快速查词弹层。
- 快速查词弹层支持发音播放与跳转完整词条。
- `WordEntryDetailFragment` 支持按 `wordId` 或 `wordText` 打开。
- `WordExamPracticeFragment` 提供与当前词关联的真题练习。
- 真题页支持题型、分类、状态筛选，以及答案展开/收起。
- `LearningDoneFragment` 汇总本轮学习结果，包括数量、时长、正确率等。
- 完成页支持继续下一轮学习，也支持点词进入词条详情。
- `LearningCheckInFragment` 会在满足条件时自动执行当日打卡并展示结果。

### 6.5 自由练习

#### 6.5.1 听力练习

- 支持从自选词、随机词或当前词书构建练习会话。
- 支持随机模式与复习队列回流机制。
- 会话内支持自动发音。
- 支持选义作答、跳过、错题回流、结果汇总。
- 会在结束时写入练习会话记录与时长统计。

#### 6.5.2 跟读练习

- 播放参考音频后可录音跟读。
- 支持回放参考音和用户录音。
- 支持调用语音评测能力获取总分、发音、流利度等结果。
- 低分或异常音频可自动进入复习轮次。
- 不支持完整评测时，页面会退化为本地占位分析并允许继续完成流程。
- 结束后写入练习会话记录。

#### 6.5.3 拼写练习

- 支持键盘和字母池输入。
- 新题自动播音。
- 支持提示。
- 每题最多两次提交机会，第二次仍错会直接展示答案并进入下一题。
- 结束后写入练习会话记录。

#### 6.5.4 音频循环

- 支持顺序播放单词，或单词 + 例句。
- 支持设置播放次数。
- 支持设置间隔秒数。
- 支持循环播放。
- 支持控制是否显示音标、中文。
- 支持上一个、下一个、播放列表、暂停、继续。
- 全部播放失败时会停止并提示。
- 完成或退出时保存练习记录。

#### 6.5.5 自选词

- 自选词页面支持搜索。
- 支持勾选多个词。
- 支持全选当前可见项。
- 支持清空选择。
- 确认时必须至少选择 1 个词。

#### 6.5.6 练习记录

- 首页练习 Tab 展示最近练习记录。
- 单条记录可查看模式、题量、完成量、正确数、提交数、时长和涉及单词。

### 6.6 账号资料与反馈

#### 6.6.1 账号资料

- 支持修改昵称。
- 支持修改性别。
- 支持修改头像。
- 头像链路支持相册选图、拍照、裁剪、预览。
- 支持修改密码；成功后会强制回到登录页。
- 支持删号确认，带 10 秒倒计时。
- 支持微信 / QQ 绑定入口。
- 若修改密码、社交绑定或头像上传接口返回 `404 / 501`，会以“敬请期待 / 即将上线”处理。
- 退出登录时，如果本地仍有未同步数据，会先弹出数据丢失风险确认，再允许强制退出。

#### 6.6.2 绑定手机号

- 页面与 ViewModel 已存在。
- 当前仅保留输入框、发送验证码、确认绑定按钮。
- 页面说明文案为“暂未开放”，点击发送或确认会 Toast“绑定手机号功能即将上线”。

#### 6.6.3 反馈

- 支持输入反馈文本。
- 支持填写联系方式。
- 支持最多上传 3 张图片。
- 单张图片最大 `5MB`。
- 图片总大小最大 `12MB`。
- 反馈文本最大 `200` 字。
- 发送中按钮会禁用。
- 成功后会清空文本、联系方式和图片列表。

#### 6.6.4 关于

- 展示应用名、版本号、使命文案。
- “检查更新”当前仅提示“暂未开放”。
- 布局里存在“去评分 / 服务条款 / 隐私政策”三行，但当前代码未绑定点击逻辑。

### 6.7 悬浮复习

#### 6.7.1 设置能力

- 支持悬浮复习总开关。
- 支持来源：当前词书 / 自选词。
- 支持顺序：随机、记忆曲线推荐、字母升序、字母降序、长度升序、长度降序。
- 支持字段显示配置：单词、音标、释义、词性、例句、笔记、图片。
- 支持调整字段字号。
- 支持卡片透明度预览。
- 支持自动随 App 启动。
- 设置变更后，如服务已开启，会按需发送 `REFRESH` 或 `PREVIEW_CARD`。

#### 6.7.2 悬浮服务行为

- 以独立 `:floating` 进程运行。
- 启动前检查悬浮窗权限；无权限则停止自己。
- 悬浮球支持拖拽。
- 悬浮球支持单击展开/收起卡片。
- 悬浮球支持拖拽自由定位，不再支持双击停靠或停靠态恢复。
- 支持根据安全区域、状态栏、导航栏、刘海重新计算位置。
- 支持记忆上次自由位置。
- 卡片支持下一个词、查看详情、关闭服务。
- 查看详情会打开完整词条页。
- 每次展示单词会记录悬浮展示记录。

### 6.8 同步、Bootstrap、下载与语音

#### 6.8.1 登录后 Bootstrap

`DataBootstrapCoordinator` 在登录后会按顺序执行：

1. 同步 Onboarding 状态。
2. 同步学习计划。
3. 同步我的词书列表，并对齐本地词书集合。
4. 同步词书进度。
5. 同步学习记录、学习时长。
6. 同步收藏。
7. 同步打卡配置、打卡状态、打卡记录。
8. 同步练习设置、练习历史。
9. 同步悬浮设置、悬浮展示记录。
10. 下载缺失的词书内容页与词状态。

额外行为：

- 远端已移除的词书会在本地删除并取消其下载任务。
- 本地未同步出去的 Outbox 内容会在拉取远端数据后再次覆盖回本地，避免本地修改被服务端旧数据抹掉。

#### 6.8.2 Sync Outbox

当前 Outbox 覆盖的业务类型包括：

- 学习记录
- 学习时长
- 练习时长
- 练习会话
- 收藏
- 词书进度
- 当前词书选择
- 单词学习状态
- 学习计划
- Onboarding 状态
- 练习设置
- 悬浮设置
- 悬浮展示记录
- 打卡记录

同步特征：

- 前台自动触发 Drain。
- 有网络且存在可重试任务时会自动重新调度 Drain Worker。
- 首页会根据待同步数、阻塞数、网络状态显示同步 Banner。

#### 6.8.3 通用下载

- 支持启动下载。
- 支持暂停下载。
- 支持继续下载。
- 支持取消下载。
- 支持通知栏交互。
- 下载完成后可根据策略打开文件或处理 APK 安装。

#### 6.8.4 语音能力

- 语音层支持阿里云与百度两套 Provider。
- 可按任务类型选择 Provider：单词 TTS、句子 TTS、跟读评测。
- TTS 带缓存与并发去重能力。
- 跟读评测供跟读练习页使用。

3. 截至本次检查，文档已覆盖当前仓库中所有主要用户可见功能、核心后台能力和已暴露但未完全落地的入口。
