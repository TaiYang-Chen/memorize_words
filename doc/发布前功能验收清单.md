# 发布前功能验收清单

> 目标：首发版本先保证“可上架、可信、可持续迭代”，不暴露未完成入口，不用付费会员文案承诺尚未接入的支付能力。

## P0 必验闭环

| 类别 | 验收项 | 通过标准 |
|---|---|---|
| 会员与权益 | 首页会员卡、会员中心、个人信息页会员状态 | 文案只表达“签到领取体验会员”；不出现默认“黄金会员”、付费开通、立即购买等误导 |
| 手机号绑定 | 个人信息页 | 使用阿里云 Fusion 本机号码认证；认证成功后调用后端绑定当前账号 |
| 登录注册 | 密码登录、注册、短信登录、一键登录、微信、QQ | 生产环境可完成；失败提示可读；不可出现 404/501 或“即将上线” |
| 账号资料 | 昵称、性别、头像、邮箱、改密、注销、退出登录 | 成功/失败/未同步数据风险提示均符合预期 |
| 法务与更新 | 官网、服务条款、隐私政策、检查更新、评分 | 所有链接真实可访问；隐私政策覆盖权限、数据同步、账号注销、第三方 SDK |
| 核心学习 | Onboarding、选词书、下载词书、设计划、新学、复习、完成页、打卡 | 新用户从安装到完成第一轮学习无阻断 |
| 练习能力 | 听力、跟读、拼写、随身听、真题、自选词、随机词 | 每个入口有数据、空数据、权限拒绝、网络失败兜底 |
| 同步与离线 | 登录后 bootstrap、Sync Outbox、前台重试、退出登录风险提示 | 离线数据恢复联网后可上传；冲突或阻塞状态有用户可理解提示 |
| 权限合规 | 通知、麦克风、相机、悬浮窗、前台服务、安装包权限 | 权限只在功能触发时申请；拒绝后可继续使用非相关功能 |

## P1 首发前建议走查

| 类别 | 验收项 | 通过标准 |
|---|---|---|
| 复习解释 | 今日新词/复习入口、练习回流 | 用户能理解今天为什么学这些词、错题为何回流 |
| 内容完整度 | 释义、例句、词根、词形、近反义、真题、发音 | 主流词书高频词不出现大面积空内容 |
| 查词与生词 | 例句内查词、完整词条、收藏夹 | 查词失败、无结果、收藏/取消收藏状态一致 |
| 发布兜底 | 无词书、无网络、下载失败、语音失败、同步阻塞、空词表 | 不出现技术堆栈、空白页、无法返回 |

## 二期竞争力池

- 学习提醒/复习提醒与可配置通知时间。
- 全局查词入口、生词本、错题本、难词本分层。
- 图片/场景记忆、真实语境例句、搭配与派生树。
- 词汇量测试、遗忘预测、个性化复习解释。
- 组队打卡、排行榜、PK、桌面小组件、跨端导出。
